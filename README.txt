   ===========================
    TemporalOperators library
   ===========================



   1. Overview

This library provides operators to manipulate web services business protocols
augmented with temporal constraints.



   2. Dependencies

The library has the following dependencies (jars included in lib/):
- Apache Commons Logging
- Apache Log4J
- Dom4J + Jaxen
- AntLR
- Protocols library.
