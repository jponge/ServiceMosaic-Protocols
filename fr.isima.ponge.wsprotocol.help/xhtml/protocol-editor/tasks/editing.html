Editing a protocol

Adding new elements

New states and operations can be added from the palette.

  • To add a state, select the type that you want in the palette then click in the editor at the desired location. It will be added with a default size. You can also draw a rectangle at the creation time to have a different initial size. Alternatively, you can drag and drop from the palette a state to the desired location and it will create a new one with the default size.
  • To add a message transition, select the type that you want from the palette, then select the source state by clicking on it. Similarly, click on the target state and the transition will be created. If the source and target states are the same, the transition will be automatically drawn as a loop.

Moving elements

  • States can be moved by selecting them and moving them to a different location.
  • Transitions can be reconnected to different source and target states. To do that, select either the source of target state connection point then drag it to the desired state.
  • To enhance the display of transitions, bendpoints can be created. Simply select the desired transition, then click on the small points that appear between existing bendpoints to create new ones. A bendpoint that is between two other ones can be deleted by aligning it on the same segment formed by the other ones. From there, release the mouse and it will be deleted.

Deleting elements

Selected elements can be deleted by using the Delete action of the context menu or the Edit menu. Pressing your usual Delete key (depending on your specification operating system / environment combination) will also delete the selected elements.

Undo / redo

It is possible to redo and undo editing operations. To do that, you can use the standard Undo and Redo actions that are available from the toolbar, the Edit menu and the editor contextual menu.

Changing properties

When items are selected, the standard Eclipse properties view will change to reflect the properties that are available for editing. For instance you can change the name of a state as well as its type.
